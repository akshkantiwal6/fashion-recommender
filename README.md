# 👕 Fashion Recommender (MERN Project)

This is a simple web app that suggests outfits based on your choices.

You can select things like weather, occasion, gender, and what clothes you already have, and it will recommend a matching outfit with an image.

## 🚀 Features
- Login & Signup (data stored in MongoDB)
- Outfit suggestions based on user input
- Image-based recommendations
- Clean and simple UI

## 🛠 Tech Used
- React (frontend)
- Node + Express (backend)
- MongoDB (database)

## ▶️ How to Run

Clone the project:
git clone https://github.com/akshkantiwal6/fashion-recommender.git

Install dependencies:
cd client  
npm install  

cd ../server  
npm install  

Run project:
cd client  
npm start  

cd ../server  
node server.js  

## 📌 Note
Make sure MongoDB is running on your system.
